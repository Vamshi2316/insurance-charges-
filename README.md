# Insurance Charges Prediction

## Problem Statement
Predict medical insurance charges based on personal and lifestyle attributes using machine learning.

## Dataset
The dataset contains features such as:
- Age
- Sex
- BMI
- Number of children
- Smoker status
- Region

Target variable:
- Insurance charges

## Approach
- Loaded and explored the dataset
- Performed data cleaning and preprocessing
- Encoded categorical variables
- Trained machine learning models
- Evaluated model performance

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Results
Built a regression model to predict insurance charges with reasonable accuracy.

## Key Learnings
- Importance of data preprocessing
- Handling categorical variables
- Relationship between features and prediction performance
